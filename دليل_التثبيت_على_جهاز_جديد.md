#  دليل التثبيت والتشغيل على جهاز جديد
1. افتح Terminal في مجلد المشروع
2. انتقل إلى مجلد backend:
   ```bash
   cd backend
   ```
3. أنشئ بيئة افتراضية:
   ```bash
   python -m venv venv
   ```
4. فعّل البيئة الافتراضية:
   ```bash
   venv\Scripts\activate
   ```

6. ثبت جميع المتطلبات:
   ```bash
   pip install -r requirements.txt
   pip install fastapi[standard]
   ```
------------------------
### الخطوة 3: إعداد Frontend (الواجهة الأمامية)

1. افتح Terminal جديد في مجلد المشروع
2. انتقل إلى مجلد frontend:
   ```bash
   cd frontend
   ```
3. ثبت جميع الحزم المطلوبة:
   ```bash
   npm install
   ```
   هذه العملية قد تستغرق بضع دقائق في المرة الأولى.

## ▶️ تشغيل المشروع
########################################## تشغيل Backend
1. افتح Terminal في مجلد المشروع
2. انتقل إلى مجلد backend:
   ```bash
   cd backend
   ```
3. فعّل البيئة الافتراضية:
   ```bash
   venv\Scripts\activate
   ```
4. شغّل السيرفر:
   ```bash
   python main.py

   ```
########################################### تشغيل Frontend
1. في Terminal جديد، انتقل إلى مجلد frontend:
   ```bash
   cd frontend
   ```
2. شغّل السيرفر:
   ```bash
   npm run dev
   ```
